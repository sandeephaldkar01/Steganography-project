Project: Steganography - Hiding Information in the Image

Description:
This Python project hides a secret message inside an image using the Least Significant Bit (LSB) technique.

Files Included:
- steg_encode.py: Script to encode a message into an image
- steg_decode.py: Script to decode the message from the image

Requirements:
- Python 3.x
- Pillow library

To Run:
1. Place your input image (e.g., input.png) in this folder.
2. Run steg_encode.py to hide a message.
3. Run steg_decode.py to extract the hidden message.
